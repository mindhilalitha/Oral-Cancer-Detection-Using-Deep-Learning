# Oral Cancer Detection Using Deep Learning

## Project Description
This project detects oral cancer from oral images using Deep Learning. Users can upload an image, and the system predicts whether it is Cancer or Non-Cancer.

## Features
- User Login and Registration
- Image Upload
- Oral Cancer Prediction
- Prediction History
- Admin Dashboard

## Technologies Used
- Python
- Django
- TensorFlow
- Keras
- OpenCV
- HTML
- CSS
- JavaScript
- SQLite

## How to Run

```bash
python manage.py runserver
```
